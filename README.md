# Steam-Idle-node
Steam Idle Trading Cards using node.js

# Installation

- [Node.js](https://nodejs.org/)
- [node-steam v0.6.8](https://github.com/seishun/node-steam)
```
npm install steam
```
- install request
```
npm install request
```

# Setting and Run

- Open _start_idle.ini and add accounts section [Accounts]
- Run _start_idle.bat
